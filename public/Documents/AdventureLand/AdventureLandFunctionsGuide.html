Adventure Land All Functions Guide

Function Description Example
smart_move(destination) Automatically navigates the character to a specific location or NPC. smart_move({to: "main"}); // Moves to the main town
get_player(name) Returns the player object with the specified name if they are visible. let player = get_player("Taro");
get_npc(name) Returns the NPC object by name if it's nearby. let npc = get_npc("fisherman");
find_npc(name) Returns the coordinates of the specified NPC, used with smart_move. smart_move(find_npc("fisherman"));
accept_quest(name) Accepts a quest from an NPC if it's available. accept_quest("cave_snakes");
complete_quest(name) Completes a quest if the objectives have been fulfilled. complete_quest("cave_snakes");
start_character(name, script) Starts a character with a specific script file. start_character("Taro", "warrior_script");
load_code(file) Loads a specific JavaScript file into the current character's runtime. load_code("combat_utils");
send_cm(target, data) Sends code message (cm) data to another character. send_cm("Healer", {type: "heal_me"});
on_cm(name, data) Handles incoming code messages from other characters. on_cm = function(name, data) { if(data.type === "heal_me") use_skill("heal", get_player(name)); };
use_skill(skill, target) Uses a skill optionally targeting an enemy or ally. use_skill("attack", get_target());
get_targeted_monster() Returns the monster your character is currently targeting. let mob = get_targeted_monster();
change_target(target) Changes the currently targeted monster or player. change_target(get_nearest_monster());
get_nearest_monster(args) Finds the nearest monster based on filter parameters. let mob = get_nearest_monster({min_xp: 100, max_att: 120});
is_on_cooldown(skill) Checks if a skill is currently on cooldown. if (!is_on_cooldown("attack")) use_skill("attack");
can_use(skill) Returns true if the skill is usable (off cooldown, has mana, etc). if (can_use("burst")) use_skill("burst");
loot() Automatically picks up nearby items. setInterval(loot, 1000);
attack(target) Attacks the specified monster or player. let target = get_nearest_monster(); if (target) attack(target);
get_monsters() Returns a list of all visible monsters. let mobs = get_monsters(); mobs.forEach(m => log(m.name));
parent.socket.emit(event, data) Sends a raw socket message to the server. Used for advanced interactions. parent.socket.emit("skill", {name: "attack", id: target.id});
buy(name, amount) Buys an item from the shop you're interacting with. buy("hpot0", 10);
sell(slot, amount) Sells the item in your inventory at the given slot. sell(0, 1);
use(item_slot) Uses a consumable item in the specified inventory slot. use(2); // Uses the item in slot 2
equip(item_slot) Equips the item in the specified inventory slot. equip(1); // Equips the item in slot 1
unequip(slot_name) Unequips an item from the specified gear slot. unequip("ring1");
game_log(message, color) Logs a colored message in the game log. game_log("Starting hunt!", "#00FF00");
clear_log() Clears the game log. clear_log();
respawn() Respawns your character after dying. respawn();
set_message(message) Sets the message displayed over your character's head. set_message("Follow me!");
change_server(region, id) Changes to a different server by region and id. change_server("US", 1);
